Projet AlgoBox, gestion de versions avec Git:



Description



Projet d'algorithmique développé avec AlgoBox dans le cadre de ma formation à l'EPF (2ème année).



Comment ouvrir les fichiers



1\. Téléchargez et installez \*\*AlgoBox\*\* : http://www.xm1math.net/algobox/

2\. Ouvrez AlgoBox

3\. Allez dans \*\*Fichier → Ouvrir\*\*

4\. Sélectionnez le fichier dans le dossier `src/`



Exécuter



\- Ouvrez le fichier `.alg` dans AlgoBox

\- Cliquez sur \*\*"Exécuter l'algorithme"\*\* (ou appuyez sur F5)
