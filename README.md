\# Projet AlgoBox - Gestion de versions avec Git



\## Description du projet



Ce projet contient des algorithmes développés avec AlgoBox dans le cadre de ma formation. L'objectif est d'apprendre à utiliser Git et GitHub pour la gestion de versions d'un projet de programmation.



\## Structure du projet

```

projet\_algobox/

├── src/

│   └── algorithme1.algo.alg    # Fichier AlgoBox principal

└── README.md                    # Ce fichier

```



\## Prérequis



Pour ouvrir et exécuter les fichiers de ce projet, vous aurez besoin de :



\- \*\*AlgoBox\*\* : logiciel de création d'algorithmes

&nbsp; - Téléchargement : \[Site officiel AlgoBox](http://www.xm1math.net/algobox/)

&nbsp; - Compatible Windows, Mac, Linux



\## Comment ouvrir les fichiers



1\. Installez AlgoBox si ce n'est pas déjà fait

2\. Lancez AlgoBox

3\. Allez dans \*\*Fichier → Ouvrir\*\*

4\. Naviguez jusqu'au dossier `src/` de ce projet

5\. Sélectionnez le fichier `algorithme1.algo.alg`

6\. Le fichier s'ouvre dans l'interface AlgoBox



\## Comment exécuter l'algorithme



1\. Une fois le fichier ouvert dans AlgoBox

2\. Cliquez sur le bouton \*\*"Exécuter l'algorithme"\*\* (ou F5)

3\. Suivez les instructions affichées dans la console d'exécution

4\. Les résultats s'afficheront dans la fenêtre de sortie



\## Informations sur la gestion de versions



Ce projet utilise Git et GitHub pour :

\- Sauvegarder régulièrement le travail

\- Suivre l'historique des modifications

\- Apprendre les bonnes pratiques de développement



\### Workflow Git utilisé

```bash

git pull origin main          # Récupérer les dernières modifications

\# ... travailler dans AlgoBox ...

git add .                     # Ajouter les modifications

git commit -m "description"   # Enregistrer les changements

git push origin main          # Envoyer sur GitHub

```
